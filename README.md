# Project Ideas

**64 real-world projects with step-by-step roadmaps.** No scaffolding. No magic. Just you, a problem, and the next tool you need.

## What is this?

A collection of project roadmaps designed for students who want to build real things from scratch. Each project starts with the absolute minimum ‚Äî an input and a button, a canvas and a click ‚Äî and grows one requirement at a time.

You don't learn tools because a syllabus told you to. You learn them because your app broke and you need to fix it.

## Every roadmap includes

- **Metadata** ‚Äî difficulty, team size, estimated time, prerequisites
- **"Built by"** ‚Äî real companies that charge money for the same thing you're building
- **Skills earned** ‚Äî specific, resume-worthy technical skills
- **Step-by-step guide** ‚Äî each step introduces one new concept, motivated by a real problem
- **Time estimates** ‚Äî realistic for students learning as they go
- **Resources** ‚Äî links to official docs, MDN, GitHub repos
- **"Where to go from here"** ‚Äî because the project never really ends
