# Web Applications

Full-stack projects that start in the browser and grow into real platforms.

| Project | Starting Point | You'll Eventually Learn |
|---|---|---|
| [Email Service](./email-service) | Send a string from A to B | SMTP, DNS, queues, spam, encryption |
| [Blog Platform](./blog-platform) | Render markdown as HTML | Rich text editing, auth, feeds, SEO |
| [Online Code Judge](./code-judge) | Run a JS function and check output | Sandboxing, containers, queues, security |
| [API Testing Tool](./api-testing-tool) | Send a fetch request and show the response | Request builder, collections, auth, environments |
| [URL Monitoring](./url-monitoring) | Ping a URL and check status code | Scheduling, notifications, time-series, dashboards |
| [Code Screenshot](./code-screenshot) | Highlight code in a div | Themes, fonts, canvas rendering, export |
| [Markdown to Slides](./markdown-slides) | Split markdown on `---` | Transitions, speaker notes, themes, PDF export |
| [Static Site Generator](./static-site-generator) | Convert one .md file to .html | Templating, file watching, live reload, asset pipeline |
| [Database GUI](./database-gui) | Connect to a DB and show tables | Query editor, results grid, multiple engines |
| [Draw DB](./draw-db) | Draw a box on a canvas | Drag-and-drop, relationships, SQL export |
| [File Diff & Merge](./file-diff) | Compare two strings character by character | Diff algorithms, side-by-side view, merge conflicts |
| [Web Terminal](./web-terminal) | Print command output in a div | PTY, WebSocket, xterm.js, sessions |
| [Web File Manager](./file-manager) | List files in a directory | Upload, rename, drag-and-drop, preview |
| [Offline-First Notes](./offline-notes) | An input that saves to localStorage | IndexedDB, sync, conflict resolution, CRDTs |
| [NST Forum](./nst-forum) | A form that posts a message | Threads, voting, moderation, search, notifications |
| [Recipes from Fridge](./fridge-recipes) | Match ingredient strings to recipes | Image recognition, API integration, ML |
