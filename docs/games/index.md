# Games

Start with a game loop. End with a game.

| Project | Starting Point | You'll Eventually Learn |
|---|---|---|
| [Tetris](./tetris) | A square that falls down | Rotation, collision, scoring, levels, speed |
| [Pokemon Open World](./pokemon) | A character that moves on a grid | Tiles, NPCs, battles, inventory, save/load |
| [Shasn The Game](./shasn) | A board with clickable spaces | Turn logic, cards, multiplayer, game state |
| [Mario Platformer](./mario) | A rectangle that can jump | Gravity, platforms, enemies, scrolling, sprites |
